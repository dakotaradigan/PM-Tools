# Research Synthesizer

This skill provides the domain knowledge for transforming raw user research into structured solution management artifacts.

## When to Use

- Raw research files (interviews, surveys, workshops) need analysis
- User asks to "synthesize", "analyze", "extract themes", or "build personas"
- Research exists but hasn't been structured into actionable artifacts

## Output Artifacts

1. **Research Summary** — Executive summary, methodology, ranked findings, theme heatmap
2. **Persona Cards** — Data-backed personas with goals, pain points, workarounds, key quotes
3. **Problem Statements** — Who/Situation/Problem/Impact format with HMW questions
4. **Theme Matrix** — Theme x Source traceability table

## Key Principles

- Every finding traces to a source. No invented data.
- Contradictions between segments are features, not bugs — surface them.
- Personas come from the data, not templates.
- HMW questions are open-ended, not solution-prescriptive.
- Flag low-confidence findings (single source, ambiguous evidence).
