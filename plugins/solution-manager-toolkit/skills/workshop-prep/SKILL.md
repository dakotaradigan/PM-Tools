# Workshop Prep

This skill provides domain knowledge for designing and preparing effective user research and discovery workshops.

## When to Use

- Planning a workshop, discovery session, or facilitated stakeholder meeting
- User asks to "prep a workshop", "create an agenda", "generate discussion prompts"
- Need structured facilitation materials before a research session

## Workshop Types Supported

- **Discovery** — Open-ended exploration of pain points and needs
- **Validation** — Testing hypotheses or problem statements with stakeholders
- **Prioritization** — Ranking and scoring features, problems, or requirements
- **Design** — Collaborative solution ideation and journey mapping

## Output Artifacts

1. **Agenda** — Timed, with activity descriptions and facilitator notes
2. **Discussion Prompts** — Open-ended questions that surface tension between roles
3. **Pre-Read** — One-page participant preparation document

## Key Principles

- Design for disagreement — consensus too early means you missed something.
- Match activities to workshop type (card sorting for discovery, dot voting for prioritization).
- Include facilitator notes (what to watch for, how to redirect tangents).
- Pre-reads stay under 1 page. Participants won't read more.
- Time-box everything. Include buffer for discussions that run long.
